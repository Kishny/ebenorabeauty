# Ebenora Beauty — site mobile-first cosmétique

Ebenora Beauty est une base complète de site vitrine/boutique mobile-first pour une marque de cosmétiques pensée pour les femmes aux carnations ébène, brunes et noires.

## Nom proposé

**Ebenora Beauty**

Idée : contraction de **Ébène** + une sonorité premium/féminine. Le nom est pensé pour être mémorisable, luxueux et cohérent avec la beauté mélaninée. À vérifier juridiquement avant dépôt de marque et achat de domaine.

## Pages / sections incluses

- Accueil premium mobile-first
- Diagnostic peau interactif
- Boutique produits
- Routines par problématique
- Skin Guide éducatif
- Présentation application mobile
- Témoignages
- Footer avec sous-pages prévues : mentions légales, CGV, contact, guide, application

## Installation

```bash
npm install
npm run dev
```

Puis ouvrir l'URL affichée dans le terminal, souvent :

```bash
http://localhost:5173
```

## Build production

```bash
npm run build
npm run preview
```

## Dépendances principales

- Vite
- React
- React DOM
- Framer Motion
- Lucide React

## À faire pour une version commerciale

- Vérifier le nom de marque et le nom de domaine.
- Ajouter de vraies photos modèles et produits.
- Connecter la boutique à Stripe, Shopify Headless, Medusa ou un backend Node/MongoDB.
- Ajouter les pages légales complètes.
- Ajouter un vrai moteur de diagnostic, avec limites médicales claires.
- Créer l'application mobile React Native/Expo à partir des mêmes écrans UX.
